核心特性

为什么选择 Deep Research Ultra

不只是搜索，而是智能分析。从关键词生成到报告输出，每一步都经过优化。

13 引擎并发搜索

中文 7 个 + 国际 5 个 + 增强 3 个，覆盖百度、必应、DuckDuckGo、Tavily、Jina 等主流搜索引擎，确保信息全面。

智能评分排序

标题相关 40 + 内容丰富 30 + 权威性 20 + 时效性 10，自动过滤低质量结果。

Claude 智能选引擎

Claude Agent 根据查询语言和主题，动态选择最佳引擎组合。不是硬编码规则，而是真正理解你的需求。

智能缓存系统

MD5 密钥 + 1 小时 TTL，相同查询直接返回缓存结果。避免重复搜索，节省时间和资源。

深度阅读能力

提取网页全文内容，Jina Reader 优先。支持 --read 参数，深入阅读目标页面，获取完整信息。

多格式输出

Markdown、JSON、报告、CSV 四种输出格式。满足不同场景需求，从快速查看到数据分析。

工作流程

从查询到报告，全自动完成

7 个步骤，自动化调研流程，让你专注于结果而非过程。

01

Claude 分析

理解查询语义，判断语言和主题，选择最佳引擎组合。

02

生成关键词

自动生成中英文关键词变体，扩大搜索覆盖。

03

并发搜索

指定引擎同时搜索，结合缓存系统，快速获取结果。

04

质量评分

标题相关 40 + 内容丰富 30 + 权威性 20 + 时效性 10。

05

去重排序

URL 去重，按质量分排序，过滤低质量结果。

06

迭代搜索

基于结果生成新关键词，继续深度挖掘。

07

输出结果

支持 Markdown、JSON、报告、CSV 四种格式。

使用示例

简单命令，强大功能

从技术选型到竞品分析，一个命令搞定深度调研。

基础搜索 
# 搜索技术问题
python search.py "2025年最新Python Web框架对比"

# 深度阅读网页
python search.py "FastAPI 最佳实践" --read

# 指定搜索引擎
python search.py "React vs Vue" --sources github,bing

# 输出格式
python search.py "AI 技术趋势" --output report
